# Инструкция по установке и использованию xxaSklad

## Требования

- **Windows 7 SP1 или выше** (32 или 64 бит)
- **.NET Framework 4.7.2 или выше**
- **Visual Studio 2019 Community Edition или выше** (для разработки)
- **MySQL Server 5.7 или выше** (или MySQL 8.0)
- **MySQL Connector/NET 8.0.33**

## Шаг 1: Установка зависимостей

### Установка MySQL Server

1. Скачайте MySQL Server с сайта https://dev.mysql.com/downloads/mysql/
2. Запустите инсталлятор и следуйте инструкциям
3. Установите пароль для пользователя `root`
4. Убедитесь, что MySQL запущена и доступна на `localhost:3306`

### Проверка подключения MySQL

```bash
mysql -u root -p
```

При подключении должна появиться приглашение MySQL.

## Шаг 2: Подготовка базы данных

1. Откройте MySQL Workbench или командную строку MySQL
2. Выполните скрипт из файла `Database/CreateDatabase.sql`:

```bash
mysql -u root -p < Database/CreateDatabase.sql
```

3. Выполните скрипт инициализации данных `Database/InitialData.sql`:

```bash
mysql -u root -p xxaSklad < Database/InitialData.sql
```

## Шаг 3: Открытие проекта в Visual Studio

1. Откройте Visual Studio 2019+
2. Выберите "Open Project or Solution"
3. Перейдите в папку проекта и выберите файл `xxaSklad.sln`

## Шаг 4: Установка NuGet пакетов

1. В Visual Studio откройте Package Manager Console:
   - Tools → NuGet Package Manager → Package Manager Console

2. Выполните команду:
```powershell
Install-Package MySql.Data -Version 8.0.33
```

## Шаг 5: Обновление конфигурации

Откройте файл `Config/App.config` и проверьте строку подключения:

```xml
<add name="MySQLConnection" connectionString="Server=localhost;Database=xxaSklad;User Id=root;Password=вашпароль;Charset=utf8;" providerName="MySql.Data.MySqlClient" />
```

**Замените `вашпароль` на пароль вашей БД MySQL.**

## Шаг 6: Запуск приложения

1. Нажмите **F5** или выберите **Debug → Start Debugging**
2. Приложение запустится с формой входа

## Учетные данные по умолчанию

При первом входе используйте:

- **Логин:** `admin`
- **Пароль:** `admin123`

Рекомендуется изменить пароль администратора после первого входа.

## Основные возможности

### 1. Управление клиентами
- Добавление новых клиентов
- Редактирование информации о клиентах
- Удаление клиентов
- Поиск по названию или телефону

### 2. Управление товарами

- Отслеживание текущих остатков

- Создание категорий товаров
- Организация товаров по группам

- Регистрация поступления товаров
- Автоматическое обновление остатков
- История всех приходов

### 5. Расход товаров (продажа)
- Регистрация отпуска товаров клиентам
- Автоматическая проверка наличия
- История всех расходов

### 6. Остатки на складе
- Просмотр текущих остатков
- Отслеживание минимальных уровней
- Предупреждения при нехватке товара

### 7. История операций
- Полная история всех действий
- Отслеживание кто, что и когда сделал

### 8. Отчеты
- Отчеты по товарам
- Отчеты по приходам/расходам
- Отчеты по клиентам
- Экспорт данных

## Решение проблем

### Проблема: "Unable to connect to MySQL"

**Решение:**
1. Проверьте, запущен ли MySQL Server
2. Проверьте правильность пароля в `App.config`
3. Убедитесь, что MySQL слушает на `localhost:3306`

### Проблема: "MySQL Connector not found"

**Решение:**
1. Откройте Package Manager Console в Visual Studio
2. Выполните: `Install-Package MySql.Data -Version 8.0.33`

### Проблема: "Database xxaSklad not found"

**Решение:**
1. Убедитесь, что выполнили скрипт `Database/CreateDatabase.sql`
2. Проверьте в MySQL:
```sql
SHOW DATABASES;
```
